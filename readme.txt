Smooth Update
-----

Simplicity2 のアップデート時に使用するテーマです。
本テーマを使用することで、ウィジェットの配置が崩れることなくアップデートを行えます。
-----

[使用方法]
1. Simplicity2 親テーマを公式サイト ( https://wp-simplicity.com/downloads/downloads2/ ) よりダウンロードする。
2. WordPress 管理画面 > 外観 > テーマより、本テーマを有効化する。
3. テーマ管理画面より、 Simplicity2 親テーマを削除する。
4. テーマ管理画面 > 新規追加 > テーマのアップロードより、ダウンロードしておいた Simplicity2 親テーマをインストールする。ただし、有効化はしない。
5. テーマ管理画面より、 Simplicity 子テーマを有効化する。

本テーマを有効化すると、 503 Service Unavailable の HTTP ステータスコードを示すメンテナンスモードに移行します。
これにより、検索エンジンは一時的なメンテナンスだと認識できます。
